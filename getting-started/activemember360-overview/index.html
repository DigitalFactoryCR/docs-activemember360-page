What Is ActiveMember360

In summary ActiveMember360 is a WordPress plugin that provides bi-directional integration between WordPress and ActiveCampaign. It is a prerequisite that to use ActiveMember360 users must have an ActiveCampaign account.

When using ActiveMember360 all data stored within ActiveCampaign is consider the master data, over and above that in WordPress. This allows you to control both access to your WordPress site and the display of content using data within ActiveCampaign e.g. by tags. It also allows you to use actions within WordPress to store information, trigger actions and automations within in ActiveCampaign.

If changes are made in WordPress either using ActiveMember360 shortcodes, or ActiveMember360 Extensions for LearnDash, WooCommerce, GamiPress etc that data is passed/stored to/in ActiveCampaign.

This includes the updating of contact fields, addition and removal of tags, starting and stopping of automations, subscribe/unsubscribe to lists, site and event tracking etc.

This allows you to create truely dynamic content based upon ActiveCampaign data and drive ActiveCampaign data collection and automations from actions within the WordPress site.

These features allow users of ActiveMember360 to create comprehensive membership sites in addition to very many other kinds of site.

ActiveMember360 controls the display/access to content and integration with ActiveCampaign on your site, it does not control the definition of content, that is controlled by other plugins, page builders or WordPress itself.

Nor does ActiveMember360 control the aesthetics of your website i.e. the branding, look and feel. that is controlled by the theme or page builder etc.

ActiveMember360 is designed to primarily work with the best in class e-commerce solutions such as WooCommerce/WooCommerce Subscriptions, ThriveCart etc.
