Installing ActiveMember360

After purchasing ActiveMember360 you were provided with a download link sent by email during the confirmation process. Before starting the ActiveMember360 installation process, please download the zip file for the plugin from that link and store it to your computer.

Alternatively you can download the plugin from the ActiveMember360 client portal at:

https://portal.digitalfactory.cr.

You access the portal using the email address provided when purchasing ActiveMember360.

Follow the steps outlined below to install ActiveMember360:

  1. Login to WordPress as an administrator.

  2. In the WordPress administrator main navigation click Plugins, Add New.

  3. Click on Upload Plugin.

  4. Click on Browse and navigate to the location where you saved the ActiveMember360 plugin zip file.

    Select that file to open.

  5. Click on Install Now.

  6. After a successful installation click Activate Plugin.
