WordPress Excerpts

WordPress Excerpts​

By default the WordPress excerpt is optional text associated only with a WordPress Post. Most of the time, it is used as the post summary. Dependent upon the theme being used excerpts can be displayed on different pages (e.g. search results, archives, etc.).

ActiveMember360 and Page/Post Excerpts​

ActiveMember360 extends WordPress excerpts to support both pages and posts.

The purpose of the Page/Post Excerpt when using ActiveMember360 is multifold. It can be used as a summary or description of a page/post, or to show ‘teaser’ text when a user does not have the permissions required to access the specific page/post.

If an excerpt is defined manually you are able to use ActiveMember360 shortcodes within the excerpt to control what type of content is shown. For example to show different content dependent upon the visitors status i.e. access level, logged in or not.

An excerpt is not required. If a user lacks the necessary permissions to access a specific page/post an excerpt will be shown if the ActiveMember360 Permission Settings, No Permissions, When access is not allowed Show Excerpt is set for that page/post.

If no excerpt is available the ‘No permissions page’ will be returned instead using the page set in ActiveMember360, Settings, Special Pages, No permissions page.

How to define Excerpts for Pages/Posts​

Excerpts are defined using any of the methods detailed below:

  • Manual
  • Use of the: Gutenberg, Layout Elements, More block Classic Editor <!–more–> tag
  • Automatically generated.

The priority for the excerpt that will be shown to a user will be:

  1. the manual one if defined
  2. else the one defined by using the More block or <!–more–> tag
  3. or otherwise the automatic one.

A manually defined excerpt provides the most flexible options for the excerpt content.

Manually​

To add a manual excerpt to a page/post, simply write one when using the WordPress Gutenberg editor in the Settings, Document, Excerpt panel. The visibility of the Excerpt panel is controlled in the Gutenberg editor by Options, Document panels, Excerpt. These settings are available by clicking the More tools and options icon in the top right of the page/post page.

If using the WordPress Classic editor as oppose to Gutenberg the Excerpt panel will normally be found beneath the content window. The visibility of the Excerpt panel is controlled in the WordPress Classic editor by screen options. These settings are available by clicking the Screen Options tab in the top right of the page/post page.

An excerpt can be as short or as long as you wish. Usually, given its purpose, a couple of sentences is fine.

When adding the excerpt manually, you may use HTML formatting and the tags will not be stripped.

You are also able to use ActiveMember360 Shortcodes and ActiveMember360 Custom Shortcodes within excerpts. Shortcodes allow for example different content dependent upon the visitors status i.e. access level, logged in or not.

Using the More Block or <!–more–> Tag​

If you place a Gutenberg, Layout Elements, More block or the Classic Editor <!–more–> tag in your page/post content an excerpt is defined as the content within the page/post prior to the More block or <!–more–> tag.

Please remember the content prior to the More block or <!–more–> tag is shown both in the excerpt and the full content of the page/post.

Automatically​

An ActiveMember360 Excerpt will be automatically generated if in ActiveMember360, Plugin Options, Excerpts, Force excerpt at all times is set to Yes.

By default, WordPress creates excerpts using the first 55 words of a page/post unless the length is specified in the ActiveMember360, Plugin Options, Excerpts, Max. length for WP-generated excerpts.

By modifying this value, you can adjust the length of your auto-generated excerpts to better suit your design layout.

Excerpt Formatting​

Typically, your excerpt will be styled by your active WordPress theme.

However ActiveMember360 does provide some options to further control the style of the excerpt in ActiveMember360, Plugin Options, Excerpts.

ActiveMember360 Excerpt Settings​

In addition to the ActiveMember360 Page/Post Excerpt that is specific to a page/post you can also include text that will be displayed after the page/post specific excerpt content.

If a post/page doesn’t have an excerpt defined, this additional text will never be displayed.

This additional text can be defined in the ActiveMember360, Plugin Options, Excerpts, Excerpts text.

If you don’t wish to display additional text leave the ActiveMember360, Settings, Excerpts, Common excerpt text panel blank.

When defining this common excerpt text you may use HTML formatting and the tags will not be stripped. You are also able to use ActiveMember360 Shortcodes and ActiveMember360 Custom Shortcodes.

ActiveMember360 also has controls to show the excerpt as a gradient and also with ‘foggy’ content.

These options are set in ActiveMember360, Settings, Excerpts, where you can also see examples of that styling.

ActiveMember360 Excerpt Visibility​

If a page/post excerpt is defined by whatever means, it will always be displayed when a user does not have the permissions required to access the specific page/post unless:

  • The ActiveMember360, Access Settings, No Permissions, When access is not allowed, Show Excerpt has been selected for the page/post but an excerpt does not exist.

  • The ActiveMember360, Access Settings, No Permissions, When access is not allowed, Default No Permissions Page or Perform redirect have been selected for the page/post.

  • The ActiveMember360, Access Settings, Permissions, Allow access for, Nobody (except admin) has been selected for the page/post. For this case the excerpt for this page/post will never be shown to anyone.

  • The ActiveMember360, Access Settings, Permissions, Excerpt only with permissions has been selected for the page/post. For this case the excerpt will be shown only to those with permissions to see the page/post.

ActiveMember360 Excerpts and Page/Post Visibility​

By default if an excerpt is visible it will always make a page/post visible.

So the page/post will appear in widgets and menus. Similarly, its’ parent, grandparent page/post etc. will also be visible if there are any.

This page/post visibility resulting from the presence of an excerpt can only be overridden by:

  • The menu item relating to the page/post being specified using a custom link menu item.

  • The ActiveMember360 Access Settings, Permissions, Nobody (except admin) or Hide from menu being selected.

  • The ActiveMember360, Modules, Widget Control settings or ActiveMember360, Modules, Menu Visibility settings determining the post/page visibility.
