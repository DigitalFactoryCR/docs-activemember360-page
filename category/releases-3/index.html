Releases

Learn about each ActiveMember360 release:

📄️ 3.4.6 (2021-06-09)

- Updated metabox.io library.

📄️ 3.4.5 (2021-06-02)

- Added ActiveMember360 Conditionals as ‘When’ options for GamiPress Rank requirements, Award required steps and Points awards and deductions.

📄️ 3.4.4 (2021-04-29)

- Changes to LearnDash Extension – LearnDash had a significant problem with LearnDash Version 3.4.0.7 which ActiveMember Version 3.4.3 addressed.

📄️ 3.4.3 (2021-04-27)

- Changes to LearnDash Extension to provide compatibility with LearnDash 3.4.0.7 and higher.

📄️ 3.4.2 (2021-04-10)

- Fixed a jQuery conflict with GamiPress.

📄️ 3.4.1 (2021-03-05)

- Added ActiveCampaign registration event tracking for mbr_genpass.

📄️ 3.4.0 (2021-02-08)

- Updated page/post access controls to now support conditionals. So now you can control access to pages/posts not just based upon login status, memberships, and tags but any condition supported by ActiveMember360 Conditionals.

📄️ 3.3.1 (2021-01-20)

- Added only/not for tags permissions option to bbPress forums.

📄️ 3.3.0 (2021-01-12)

- Datatables files now hosted on the server install to assist with GDPR compliance.

📄️ 3.2.9 (2020-12-01)

- Improvements to conditionals infrastructure.

📄️ 3.2.8 (2020-10-05)

- Support for latest OptimizePress 3.

📄️ 3.2.7 (2020-09-29)

- Replaced jQuery Tokenize library with jQuery select2.

📄️ 3.2.6 (2020-09-03)

- Correction for LearnDash course and lesson quiz protection.

📄️ 3.2.5 (2020-09-02)

- Compatibility enhancements to support WordPress 5.5. and legacy WordPress 5.x versions for emailing.

📄️ 3.2.4 (2020-08-12)

- WordPress 5.5. compatibility.

📄️ 3.2.3 (2020-08-10)

- Added support latest ActiveCampaign site tracking code.

📄️ 3.2.2 (2020-07-22)

- Fixed an error with lost password redirect.

📄️ 3.2.1 (2020-07-15)

- Enhancements to ActiveMember360 Conditionals:

📄️ 3.2.0 (2020-07-01)

- Added new ActiveMember360 Conditionals, see Conditionals documentation.

📄️ 3.1.9 (2020-06-08)

- Added redirect parameter for [mbr_password_change] shortcode and more styling control. [fundamentals-conditionals].

📄️ 3.1.8 (2020-05-26)

- Improvements to ActiveMember360 Menu Visibility settings to include logged in user status, and the selection of tag criteria directly, in addition to the previous and retained PHP conditional statement.

📄️ 3.1.7 (2020-05-13)

- Improvements to password send.

📄️ 3.1.6 (2020-05-04)

- Compatibility enhancements for latest WooCommerce.

📄️ 3.1.5 (2020-04-06)

- Compatibility enhancements for WordPress 5.4.

📄️ 3.1.4 (2020-01-31)

- Gutenberg Extension: Enhancement to include WordPress version check upon activation of the extension.

📄️ 3.1.3 (2020-01-27)

- Gutenberg Extension: To control Gutenberg block display. For more information visit ActiveMember360 Block Editor Module

📄️ 3.1.2 (2019-12-18)

- Modified LearnDash Extension to provide ongoing support for course access data being stored in user meta.

📄️ 3.1.1 (2019-12-11)

- For Learndash courses, “redirect when forbidden” feature has priority over displaying excerpts to support the creation of course sales pages.

📄️ 3.0.0 (2018-04-16)

- This releases contains a number of fixes for exotic issues mostly related to third party plugin and theme compatibility.

📄️ 3.0.9 (2019-11-12)

- GamiPress integration: For more information see ActiveMember360 GamiPress Extension.

📄️ 3.0.8 (2019-09-19)

- Oxygen Builder integration: Support of tag based display conditions.

📄️ 3.0.7 (2019-09-13)

- Payment Module: Improved compatibility with latest Stripe data structure.

📄️ 3.0.6 (2019-08-19)

- Learndash integration to provide compatibility with short course description data structure changes.

📄️ 3.0.5 (2019-08-13)

- Payment Module: Enhancement to Stripe integration to provide improved API compatibility.

📄️ 3.0.4 (2019-08-09)

- Enhancements to ActiveMember360 Menu Visibility functions to improve compatibility with other third party plugins.

📄️ 3.0.3 (2019-07-24)

- Elementor Popup Builder support: Our Elementor integration now supports the popup builder’s “advanced rules”. This lets you display a popup based on login status, tags and fields.

📄️ 3.0.2 (2018-11-16)

- New Elementor integration for conditional display of sections and elements.

📄️ 3.0.1 (2018-11-03)

- Minor improvements.

📄️ 3.0.0 (2018-04-16)

- This releases contains a number of fixes for exotic issues mostly related to third party plugin and theme compatibility.
