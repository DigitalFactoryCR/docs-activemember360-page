Action Shortcodes

Learn how to use ActiveMember360 Action Shortcodes:

📄️ mbr_automation

The ActiveMember360 shortcode mbr_automation adds and/or removes an ActiveCampaign contact to/from ActiveCampaign automations.

📄️ mbr_contact_count

The ActiveMember360 shortcode mbr_contact_count returns a count of the number of contacts with the specified ActiveCampaign tag.

📄️ mbr_cookie_set

The ActiveMember360 shortcode mbr_cookie_set sets a client-side cookie variable to a given value.

📄️ mbr_date

The ActiveMember360 shortcode mbr_date returns a specified date formatted and modified as requested.

📄️ mbr_datediff

The ActiveMember360 shortcode mbr_datediff returns the difference between two specified date formatted as requested.

📄️ mbr_email_change

The ActiveMember360 shortcode mbr_email_change returns a HTML form through which the current logged in user linked to an ActiveCampaign contact can change their email address.

📄️ mbr_exec

The ActiveMember360 shortcode mbr_exec allows the addition of PHP code into a post, page, excerpt or widget.

📄️ mbr_foreach

The ActiveMember360 shortcode mbr_foreach performs a set of repetitive actions a fixed number of times, based on values passed to it, similar to a programming FOREACH loop.

📄️ mbr_forloop

The ActiveMember360 shortcode mbr_forloop performs a set of repetitive actions a fixed number of times, similar to a programming FOR…TO…NEXT loop.

📄️ mbr_form

The ActiveMember360 shortcode mbr_form embeds and displays an ActiveCampaign form in your page/post.

📄️ mbr_include

The ActiveMember360 shortcode mbr_include includes the content of the file specified.

📄️ mbr_include_once

The ActiveMember360 shortcode mbr_include_once includes the content of the file specified but the file can only be included once.

📄️ mbr_jdelay

The ActiveMember360 shortcode mbr_jdelay applies a jQuery “animation” against an existing CSS ID or one created automatically.

📄️ mbr_list

The ActiveMember360 shortcode mbr_list subscribes/unsubscribes an ActiveCampaign contact to/from ActiveCampaign lists.

📄️ mbr_log_event

The ActiveMember360 shortcode mbr_log_event sets a client-side cookie variable to a given value.

📄️ mbr_login_form

The ActiveMember360 shortcode mbr_login_form returns a HTML form through which a visitor can log in to the site.

📄️ mbr_math

The ActiveMember360 shortcode mbr_math performs a calculation using the value stored in any contact field in your ActiveCampaign application.

📄️ mbr_member_dir

The ActiveMember360 shortcode mbr_member_dir updates the ActiveCampaign contact record associated with the logged in WordPress user.

📄️ mbr_my_account

The ActiveMember360 shortcode mbr_my_account returns a form through which a member can update their ActiveCampaign contact data.

📄️ mbr_page_list

The ActiveMember360 shortcode mbr_page_list returns an HTML link list of pages.

📄️ mbr_password_change

The ActiveMember360 shortcode mbr_password_change returns a HTML form through which the current logged in user linked to an ActiveCampaign contact can change their password.

📄️ mbr_password_send

The ActiveMember360 shortcode mbr_password_send returns a HTML form through which the visitor can request an email reminder of their password.

📄️ mbr_populate

The ActiveMember360 shortcode mbr_populate will populate content placed between the shortcode opening and closing tags by replacing any merge fields with ActiveCampaign contact data.

📄️ mbr_popup_inline

The ActiveMember360 shortcode mbr_popup_inline returns a HTML link which triggers a WordPress ThickBox popup window displaying the code embedded between the shortcodes opening and closing tags.

📄️ mbr_popup_inliner

The ActiveMember360 shortcode mbr_popup_inliner returns a HTML link which triggers a responsive popup window displaying the code embedded between the shortcodes opening and closing tags.

📄️ mbr_popup_page

The ActiveMember360 shortcode mbr_popup_page returns a HTML link which triggers a WordPress ThickBox popup window containing the content loaded from the specified web URL.

📄️ mbr_progress_bar

The ActiveMember360 shortcode mbr_progress_bar returns a HTML5 <progress> tag based upon a percentage calculated from the number of tags assigned to the contact linked to the logged in user compared to a specified list of available tags.

📄️ mbr_progress_meter

The ActiveMember360 shortcode mbr_progress_meter returns a HTML5 <meter> tag based upon the number of tags assigned to the contact linked to the logged in user compared to a specified list of available tags.

📄️ mbr_progress_tags

The ActiveMember360 shortcode mbr_progress_tags returns a progress factor either as; a percentage of the total, fraction of the total, a rounded division, or simple counter.

📄️ mbr_randint

The ActiveMember360 shortcode mbr_randint returns a random integer.

📄️ mbr_redirect

The ActiveMember360 shortcode mbr_redirect redirects a visitor to another page/page within the site or an external URL.

📄️ mbr_related_posts

The ActiveMember360 shortcode mbr_related_posts retrieves and displays an HTML link list of related posts.

📄️ mbr_select

The ActiveMember360 shortcode mbr_select returns a value from a specified list, based upon a switch variable, similar to a programming switch construct.

📄️ mbr_tag

The ActiveMember360 shortcode mbr_tag adds and/or removes specified tags, to/from either the logged in user contact record in ActiveCampaign, or to/from the contact specified.

📄️ mbr_texpand

The ActiveMember360 shortcode mbr_texpand generates a link to expand/collapse the specified content.

📄️ mbr_update_contact

The ActiveMember360 shortcode mbr_update_contact updates the ActiveCampaign contact record associated with the logged in WordPress user.
