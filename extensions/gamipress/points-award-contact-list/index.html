Add New Points Award Based Upon ActiveCampaign Contact List/Lists

  1. When editing a point type within the Automatic Points Award panel click Add New Points Award.
  1. Click on the When dropdown, scroll, and in the ActiveMember360 section click to select Contact is subscribed to list(s).
  2. In the dropdown select whether the user should be subscribed to either Any or All of the lists specified to earn the award.
  3. In the dropdown select the ActiveCampaign list(s) the user should be subscribed to.
  4. In the Earn input field specify the number of point types to be awarded when the contact is subscribed to the required list(s). This type of ActiveMember360 award can only be earnt once if the logged in user satisfies the contact list condition.
  5. Enter your own label. This is not compulsory and will be automatically created by GamiPress and ActiveMember360 if left blank. When subsequently editing the Automatic Points Awards the label is not automatically updated it must be edited manually.
  6. Click Save All Points Awards.
