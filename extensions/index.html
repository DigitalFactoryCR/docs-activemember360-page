Extensions

Overviewâ€‹

ActiveMember360 is compatible with very many third party WordPress plugins.

ActiveMember360 does not require an extension to work with many of those WordPress plugins.

However for some third party plugins ActiveMember360 provides a very comprehensive level of integration.

For those plugins ActiveMember360 provides an extension which must be installed and activated, along with the third party plugin itself, to unlock all features of the integration.

Available Extensionsâ€‹

Important

ActiveMember360 does not require an extension to work with many/most WordPress plugins.

Please review the extensions available and enable those as required:

  • bbPress
  • Beaver Builder & Beaver Themer
  • BuddyPress
  • Contact Form 7
  • Elementor
  • GamiPress
  • LearnDash
  • LifterLMS
  • Memberoni
  • Oxygen Builder
  • Popup Maker
  • WooCommerce
  • WP Courseware
Note

In all cases the third party plugin must be installed and activated before the associated ActiveMember360 extension can be activated.
