Quiz

Note

The ActiveMember360 LMS is currently in beta.

Add a new quiz​

  1. In the WordPress Admin backend's main navigation click AM360 LMS -> Settings.
  2. Click on Add quiz.

Basic quiz settings​

  1. Define the quiz's questions and answers. You can add as many questions and answers as you want.
  2. See connected lessons. A quiz can be connected to multiple lessons. Each connected lesson can require a different passing grade for the same quiz.
  3. Define quiz settings as detailed in the following steps.

Enhanced settings​

  1. Define the percentage of correct answer to pass the quiz.
  2. Enable limit failed attempts.
  3. Define the maximum failed attempts if enabled under (2). If the maximum failed attempts limit is reached, the user cannot complete the associated lesson with the consequence of not being able to completing the course.

Quiz passed settings​

  1. Select the tags which should be assigned to the ActiveCampaign contact when passing a quiz.
  2. Select the tags which should be removed from an ActiveCampaign contact when passing a quiz.
  3. Select a location to which the user should be redirected when passing the quiz. Default: The defined lesson completed redirect location of the related lesson.
  4. Define parameters to be added to the redirect location (HTTP GET).

Quiz not passed settings​

  1. Select the tags which should be assigned to the ActiveCampaign contact when failing a quiz. (Added after the maximum failed attempts has been reached.)
  2. Select the tags which should be removed from an ActiveCampaign contact when failing a quiz. (Removed after the maximum failed attempts has been reached.)
  3. Select a location to which the user should be redirected when failing the quiz. Default: no redirect.
  4. Define parameters to be added to the redirect location (HTTP GET).

Failed attempts​

  1. Select the tags which should be assigned to the ActiveCampaign contact after each failed attempt. If the maximum number of failed attempts is enabled, the last failed attempt is not included but triggers instead the settings under Quiz not passed -> Add tags
  2. Select the tags which should be removed from an ActiveCampaign contact after each failed attempt. If the maximum number of failed attempts is enabled, the last failed attempt is not included but triggers instead the settings under Quiz not passed -> Remove Tags
