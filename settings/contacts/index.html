Show Contact Infoâ€‹

By selecting this tab you can retrieve any contact record from ActiveCampaign by submitting an email address or ActiveCampaign contact ID.

The contact record shown is the most current live data, it is not cached data.

This provides an easy way to see a contact's live ActiveCampaign data without having to login your ActiveCampaign account. Ideal for checking what tags a contact has or their contact field values.

Change Contact Emailâ€‹

Recommendation

This method should always be used by an administrator to change the user and contact email address rather than directly editing the contact record in ActiveCampaign and editing the WordPress user profile.

By selecting this tab you can change the email address of a user and associated ActiveCampaign contact.

Using this method updates the email address both in WordPress and ActiveCampaign resulting in a new contact primary email address in ActiveCampaign and a new username and email address in WordPress.

As this method updates existing users and contacts rather than creating new ones all contact and user data is retained.

Note

No email notification of a change in email address is sent to the user/contact using this method.

Change Contact Passwordâ€‹

By selecting this tab you can change the password of a WordPress user and associated ActiveCampaign contact.

Using this method updates the password both in WordPress and ActiveCampaign.

Note

No email notification of a change in password is sent to the user/contact using this method.
