Templates

Depending upon your configuration ActiveMember360 will use a maximum of three email templates to send emails from WordPress using the standard WordPress wp_mail function.

The templates and their purpose are:

  • Password Reminder Email - Sent when using the default ActiveMember360 password configuration and the WordPress Lost Password link /wp-login.php?action=lostpassword or the shortcode [mbr_password_send]
  • Login Credentials Email - Sent by ActiveMember360 Payments module after a successful purchase where a new WordPress user is created or when both ActiveMember360, Settings, Passwords, Switch to encrypted password storage in ActiveCampaign and ActiveMember360, Modules, Automatic Webhooks are enabled
  • New User Email - Sent when a new user registers using the standard WordPress registration process

Password Reminder Email​

  1. Specify the From Name name.

    This takes preference over the Default "Mail From" name specified in ActiveMember360, Settings, Mailing, Default "Mail From" name.

    Default: As set in ActiveMember360, Settings, Mailing, Default "Mail From" name

  2. Specify the From Mail address.

    This takes preference over the Default "Mail From" address specified in ActiveMember360, Settings, Mailing, Default "Mail From" address.

    Default: As set in ActiveMember360, Settings, Mailing, Default "Mail From" address

  3. Specify the Subject for the email.

    Default: [xxx] Your username and password info

    where xxx is as defined in WordPress, Settings, General, Site Title.

  4. Specify the Message for the email.

    You can use the following placeholders in your email message/template:

    • %login_url%
    • %user_login%
    • %user_password%

    Default:

    Login at: %login_url%
    Username: %user_login%
    Password: %user_password%

  5. Click Save Settings to ensure your template changes are saved.
