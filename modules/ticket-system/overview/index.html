Ticket System Module

Overview​

The ActiveMember360 Ticket System module is for those who wish to provide a support ticket system using WordPress.

The Ticket System module is designed to be available for any logged in user/contact of your WordPress site. The Ticket System module can be hosted on a standalone site that doesn't utilise the other features of ActiveMember360.

It is not an email based ticketing system. It is designed for ticket submissions and replies to be accessed through your WordPress site. All ticket data is stored within WordPress, however the ticket submissions and replies can also be stored within ActiveCampaign as contact notes.

Whilst ticket submissions and replies are not sent by email the Ticket System module supports email notification of ticket replies. Browser based notifications are also supported.

If using WooCommerce a link to the Ticket System can be automatically configured and placed within the WooCommerce My Account page/navigation.
