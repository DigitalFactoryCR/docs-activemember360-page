Mailer Module

Overview​

The purpose of the ActiveMember360 Mailer module is for those who wish to send emails from ActiveCampaign automations using their own SMTP mail provider rather than ActiveCampaign.

There could be many reasons not to use the native ActiveCampaign email service for certain emails.

SMTP services which can be used with the ActiveMember360 Mailer module include but are not limited to:

  • Amazon SES
  • Gmail
  • Outlook
  • Mailgun
  • Mailjet
  • Mandrill
  • Moosend
  • Postmark
  • SendGrid
  • Sendinblue
  • SMTP.com
  • SparkPost

The ActiveMember360 Mailer module can use any chosen ActiveCampaign email template for sending the email.

The ActiveMember360 Mailer module will control the unsubscribe link in any email it sends and the unsubcribe/optout functions.

The sending of an email to a specific ActiveCampaign contact using the ActiveMember360 Mailer module is achieved by using a webhook within an ActiveCampaign automation.
